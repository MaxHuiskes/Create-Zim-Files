# How the ZIM Creation Script Works

## Overview
This script downloads a website and converts it into a ZIM file compatible with Kiwix.

## Step 1: Prepare Script
- Save `make_zim.sh` to your system.
- Make it executable: `chmod +x make_zim.sh`.

## Step 2: Run Script
```bash
sudo bash make_zim.sh <site_url> <output_folder> <zim_title> <zim_description> <creator>
